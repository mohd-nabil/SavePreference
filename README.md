# SavePreference
Android Shared Preference

Add it in your root build.gradle at the end of repositories:

	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
Step 2. Add the dependency

	dependencies {
	        implementation 'com.github.mohd-nabil:SavePreference:bbb75b3f02'
	}
